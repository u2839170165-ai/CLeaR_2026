# CLeaR_2026
Data, code and results used in 'Consensus Enhances Individual Causal Models: a Use Case on Lung Cancer Driven by Cellular Pathways'
This repository contains the code and datasets used in the experiments
of the submitted paper.
One can obtain plots and results by executing stats_and_visualisation.py

The repository is provided anonymously for review purposes and will be
de-anonymized upon acceptance.
